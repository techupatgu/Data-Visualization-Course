# Introduction to data visualization

This folder contains the course in data visualization given by members of the Gothenburg
Research Infrastructure in Digital Humanities at the Department of Political Science (University of
Gothenburg, Sweden), from November 3rd to 6th. The course was taught by Dr. Jonathan Westin, 
Dr. David Alfter, and Dr. Matteo Tomasini. 

The lectures by Dr. Tomasini were prepared using Quarto. To access the lectures, access the folder of 
interest and look for a file in the format `.html`. Open it in a software that reads HTML files (e.g. 
Google Chrome).
